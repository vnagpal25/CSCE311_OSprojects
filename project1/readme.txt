main.cc:
contains the main program, stack for operands/operators,
logic for reading expression as command line arguments as well as 
evaluating expression using stack

calculator.cc:
contains the implementations of helper functions that are used to 
do tasks like evaluate a simple operation, determine order of 
operations, as well as validate input

calculator.h:
contains the definitions of helper functions that are used to 
do tasks like evaluate a simple operation, determine order of 
operations, as well as validate input

makefile:
contains files to compile source files into executable called calculate,
also removes intermediary files
